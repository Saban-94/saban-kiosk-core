# Saban Kiosk Core System 🏗️

מערכת קיוסק חכמה לייעוץ ומכירת חומרי בניין.
המערכת מיועדת למסכי מגע, כוללת יועץ AI, קטלוג מוצרים והדפסת בונים.

## 🌟 פיצ'רים מרכזיים
- ממשק מותאם למסכי מגע (Kiosk Mode).
- יועץ חכם מבוסס AI (Google Gemini).
- קטלוג מוצרים דינמי (Firebase Firestore).
- ללא עלויות תפעול (Serverless).

## 🛠️ טכנולוגיות
- **Frontend:** HTML5, CSS3, Vanilla JavaScript.
- **Backend:** Firebase (Hosting, Firestore).
- **AI:** Google Gemini API.

## 🚀 איך להריץ מקומית
1. וודא שמותקן `firebase-tools`.
2. הרץ `firebase serve`.
